Mike's Notes

I put together this sample based on the mockup Jess sent around but with some changes based on the practicalities of web pages as opposed to design on a page:

Width set to standard 960px (suitable for 90% of users), info blocks at bottom would not be seen or ignored so moved to side block, side block uses 'accordian' pattern (needs better styling), removed obsolete >> markers on links - everyone knows a link takes you to a new page, scroller is automated but will pause if you hover over it, primary navigation at the top fakes being used for demo purposes, I don't have the font for the funky ticked 'i' in CompliSpace, I couldn't find the image used for the header but found this one as closest, the footer will push to the bottom of the browser window even when the content is empty.
